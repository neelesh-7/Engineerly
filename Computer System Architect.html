Overview

What do Computer System Architects Do?

  Computer system architects design and develop solutions to complex applications problems, system administration issues, or network concerns. They evaluate existing systems to determine effectiveness, and suggest changes to meet organizational requirements. Computer system architects collaborate with engineers or software developers to select appropriate design solutions or ensure the compatibility of system components, identify system data, hardware, or software components required to meet user needs. They then must direct the analysis, development, and operation of complete computer systems. Computer system architects also provide technical guidance or support for the development or troubleshooting of systems and train system users in system operation or maintenance.

  Computer system architects work indoors working on computers. They typically sit at a desk for the majority of the day, which can cause strain on one's eyes and back.
